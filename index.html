Our Project

Current system for delivery of products is a paper-based system, though some of the companies use different methods. We came to an understanding of the problems faced by them. In the existing system a list of addresses will be given to the delivery boys and they have to use google map in order to find the directions to the destined location. This is a time-consuming method. Our application is proposed in order to bring a change to this situation. Fast Finder application is created in such a way that the delivery boys can simply list out the addresses and the application connect it to google map via GPS in order to find the and sort the addresses according to the shortest distance. Sorting is done using Travelling Salesman Problem.

  • Date: October 2019
  • Service: Android Application
